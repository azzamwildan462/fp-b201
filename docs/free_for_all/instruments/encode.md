# Encode Instruments

## Urutan alat musik: 

1. Bass
2. Gitar
3. Keyboard
4. Drum
5. Vocal
6. Violin
7. String
8. Piano

## Tata cara
Perhitungan index biner dimulai dari kiri. Setiap index mengindikasikan alat musik sesuai urutan diatas. Index dimulai dari 0. Alat musik yang ingin dicari diberi nilai 1 dan sisanya diberi 0.

## Contoh
Gitar-Bass-Drum   
2     1    4    
110100000   


Keyboard   
3   
001000000   


Piano-Vocal   
8     5   
00001001
